# 正文

所谓内置数据类型的扩展，就是利用组合或继承等关系，给 Python 的内置数据类型添加功能从而支持其它类型的数据结构，比如为列表增加队列的插入和删除、限制输入范围的方法。

这里有两种方法可以实现内置类型的扩展：

- **利用组合关系实现：**创建一个新对象，并保存一个容器对象作为属性，适用于只想要用容器存储一些对象并利用它的一些特征的情况；
- **利用继承关系实现：**从 Python2.2 开始，所有内置类型都能直接创建子类，如 list，str，dict 以及 tuple。我们可以通过创建内置对象的子类并添加方法的方式来实现我们想要的操作，适用于需要修改容器对象实际的运作方式的情况。例如，如果想要确保 list 中的所有元素都必须是由 5 个字符组成的字符串，需要继承 list 并重写 append()方法，对于不合法的输入抛出异常。

## 利用组合关系实现内置数据类型的扩展

在以下示例中，通过将 list 对象作为类内属性，实现集合对象，示例代码如下：

```python
from typing import List
class Set:
    def __init__(self, value:List[int]=[]):
        _# 创建一个列表_
        self.data = []
        _# 拼接value列表_
        self.concat(value)

    _# 求交集_
    def intersect(self, other:List[int]):
        _# 创建空的临时列表_
        res = []
        for x in self.data:
            _# 如果x是相同元素，则插入空列表_
            if x in other:
                res.append(x)
        _# 返回一个新集合_
        return Set(res)

    _# 求合集_
    def union(self, other:List[int]):
        _# res为self.data列表的副本_
        res = self.data[:]
        for x in other:
        _# 对于在other列表中的x元素_
            if not x in res:
                _# # 如果x元素不在res列表中，则插入res列表中_
                res.append(x)
        _# # 返回一个新集合_
        return Set(res)

    _# 合并方法_
    def concat(self, value:List[int]):
        for x in value:
            if not x in self.data:
                _# 如果x元素不在data列表中，则插入data列表中_
                self.data.append(x)

    _# 以下为重载方法_
    _# __len__求其长度_
    def __len__(self):
        return len(self.data)  _# len(self)_

    _# __getitem__ 方法的作用是通过索引或切片来获取对象的元素，并返回对应的值_
    def __getitem__(self, key):
        return self.data[key]

    _# 运算符&重载，求交集_
    def __and__(self, other):
        return self.intersect(other)

    _# 运算符|重载，求合集_
    def __or__(self, other):
        return self.union(other)

    _# 当直接打印类的实例化对象时，系统将会输出对象的自我描述信息_
    def __repr__(self):
        return 'Set:' + repr(self.data)

if __name__ == '__main__':
    x = Set([1, 3, 5, 7])
    y = Set([1, 3, 6, 9])
    print(x, y, "len x : ",len(x), "len y : ",len(y))
    _# 求合集_
    print(x.union(Set([1, 4, 7])))
    _# 求交集_
    print(x.intersect(Set([1, 4, 7])))
    _# 求x和y的交集_
    print(x & y)
    _# 求x和y的合集_
    print(x | y)
```

可以看到，我们在这段代码中利用内置列表实现了集合的创建、求交集、求并集、合并等操作，你可能会对重载方法感觉到困惑，不过不用慌，我们接下来就会讲到，首先验证一下代码，运行结果如下，可以看到没有什么问题：

## 利用继承关系实现内置数据类型的扩展

实际上，如果需要修改内置类中的任何方法（包括魔法方法），那么应该用继承关系，上述用类内置列表属性实现集合的例子如果用继承关系写，示例代码如下：

```python
from typing import List

class Set(list):
    def __init__(self, value:List[int]=[]):
        _# 初始化一个空列表/父类初始化_
        list.__init__([])
        _# # 拼接value列表_
        self.concat(value)

    def intersect(self, other):
        res = []
        for x in self:
            if x in other:
                res.append(x)
        return Set(res)

    def union(self, other):
        res = Set(self)
        res.concat(other)
        return res

    def concat(self, value):
        for x in value:
            if not x in self:
                self.append(x)

    def __and__(self, other):
        return self.intersect(other)

    def __or__(self, other):
        return self.union(other)

    def __repr__(self):
        return 'Set:' + list.__repr__(self)
```

可以看到，利用继承关系时我们可以在子类中直接调用父类的魔法方法而无需重写。
